# Cerebro Education Platform

A Next.js frontend for the cerebro-agent-network decentralized education streaming and scholarship platform on Stellar.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the platform.

## Features

- Course access streaming interface
- Scholarship management dashboard
- Student progress tracking
- Teacher payment management
